# 💰 Expense Tracker API

A simple RESTful API for tracking personal expenses, built with Python and Flask.  
This project is designed for learning backend development, API design, and Git/GitHub workflow.

---

## 🚀 Features

- Add new expenses
- View all expenses
- Get expense by ID
- Update expense
- Delete expense
- Simple in-memory / database-ready structure

---

## 🛠 Tech Stack

- Python 3
- Flask
- REST API
- Git & GitHub
